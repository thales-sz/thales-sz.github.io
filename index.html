TRYBE FUTEBOL CLUBE

TABELA DE CONTROLE DE PARTIDAS DE FUTEBOL

O Trybe Futebol Club é um projeto fullstack que oferece uma aplicação completa de controle de partidas e tabela do futebol brasileiro. A criação do backend foi feita utilizando o Sequelize, TypeScript como linguagem, ExpressJS e validações com Joi, JsonWebToken e BcryptJS para garantir a segurança dos dados. Com o Trybe Futebol Club, os usuários podem desfrutar de uma aplicação completa e fácil de usar para acompanhar todas as partidas do futebol brasileiro.
